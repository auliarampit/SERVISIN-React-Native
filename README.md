## Required

that is required to be installed
- React-native-cli
- NPM (Node Package Manager)
- Emulator

## Clone or download
- `$ git clone https://github.com/auliarampit/SERVISIN-React-Native.git`
- `$ cd SERVISIN-React-Native`
- `$ npm install`

## Start Application
  `$ react-native run-android`
